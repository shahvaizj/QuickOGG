# QuickOgg 🎧

QuickOgg is a simple and fast Windows tool that converts audio files into the OGG format. Designed for developers, game creators, and audio engineers who need quick batch or single conversions without heavy software.

---

## 🚀 Features

- ⚡ Fast audio conversion to `.ogg`
- 📁 Supports multiple input formats (wav, flac)
- 🖱️ Drag & drop support
- 📦 Batch conversion
- 🪶 Lightweight and portable
- 🔧 No complex setup required

---

## 🖥️ Supported Formats

### Input:
- WAV
- FLAC

### Output:
- OGG (Vorbis)

---

## 📥 USAGE

1. Download the latest release from the GitHub repository's "Releases" section.
2. Extract the downloaded ZIP file to a folder of your choice.
3. Double-click on "audio_converter.exe" to launch the application.
4. Use the interface to:
   - Add files via drag-and-drop or the "Add Files" button
   - Select an output folder using the "Browse" button
   - Click "Convert" to start the conversion process
5. The application will convert all supported audio files (WAV, FLAC, OGG) to OGG format.
6. Progress and status will be displayed in the status bar at the bottom.
